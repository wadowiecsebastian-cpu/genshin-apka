# Genshin app

Static HTML/CSS/JS app prepared for GitHub Pages.

## GitHub Pages setup

1. Upload all files from this folder to the repository root.
2. Open repository Settings.
3. Go to Pages.
4. Set Source to "Deploy from a branch".
5. Select branch "main" and folder "/root".
6. Save and wait for the published URL.

The app starts from `index.html`.
